# K:Dummy - Klaviyo Dummy Data Generator

Quickly create personalised dummy data for demo and educational purposes.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- **Home** (`/`) - Main landing page with navigation
- **Profiles** (`/profiles`) - Create and manage dummy profiles
- **Events** (`/events`) - Generate events from templates or create custom events
- **Objects Catalog** (`/catalog`) - Upload and manage object data (products, locations, reservations)

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- JavaScript (no TypeScript)
